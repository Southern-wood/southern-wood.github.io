# southern-wood.github.io

Static personal homepage for `southernwood.space`.

## Structure

- `index.html`: deployed homepage
- `reading.jpg`: portrait image used by the homepage
- `CNAME`: GitHub Pages custom domain
- `docs/plans/`: local planning notes for the page
- `page-story-index.md`: source content snapshot used to draft the page

## Deployment

This repository is intended to be published directly with GitHub Pages from the repository root.
